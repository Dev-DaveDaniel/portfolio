Connect2

Connect2 is a mobile turn based game built on the rules of 'Connect 4'

This is my own personal project that I worked on from concept to prototype. I added the math element from 2048 to the game to lengthen the game time and level of difficulty. I built this game keeping in mind that it was to be played on mobile platforms.

I spent 10 days developing the project and learned these key skills:

  • Mobile Game Developtment
  • Bring an additional elemet to an already polised game.
  • Design and prototype a game mode.
  • Document my idea and thought process behaind the game
  • Playtest and bugfind other maps.
